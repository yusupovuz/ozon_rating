# Ozon Rating Checker (Clean Architecture)

Асинхронный скрипт для получения рейтинга товара с маркетплейса Ozon по его артикулу. 
Реализован **без использования парсеров (BeautifulSoup4) и браузерных автоматизаторов (Selenium)**. Вся логика построена исключительно на базе `aiohttp` и регулярных выражений (`re`) с соблюдением принципов модульной архитектуры.

## Технические решения (Обход защиты)
1. **Обход WAF (Cloudflare):** Имитация реального пользователя путем передачи валидных сессионных `Cookies` и заголовков браузера (`Headers`).
2. **Обход JS-Redirect:** Прямые запросы на региональный домен `uz.ozon.com` для избежания принудительного редиректа на стороне клиента через JavaScript (поскольку aiohttp не рендерит JS).
3. **Regex Parsing:** Парсинг сырого HTML-кода и извлечение данных напрямую из скрытых JSON/SEO состояний страницы (state/schema/microdata).

## Структура проекта
* `config.py` — конфигурационный файл (хранение Headers и Cookies).
* `ozon_client.py` — слой взаимодействия с сетью и бизнес-логика поиска данных (парсинг).
* `main.py` — точка входа в приложение (UI/Terminal interaction).
